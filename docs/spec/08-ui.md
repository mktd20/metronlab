# 08. UI/UX Guidelines

## Design Principles

- **Dark Theme** 기본 (연습 시 눈 피로 감소)
- Light Theme 전환 지원
- 그래프/차트 중심의 데이터 시각화
- 최소한의 클릭으로 연습 시작 가능
- 반응형 (Desktop, Tablet, Mobile)

## Color Palette

```
Primary:      #6C5CE7 (Purple - 메인 브랜드 컬러)
Secondary:    #00B894 (Green - 성공/달성)
Accent:       #FD79A8 (Pink - 강조)
Background:   #0A0A0F (Dark)
Surface:      #1A1A2E (Card/Panel)
Text Primary: #FFFFFF
Text Muted:   #8B8BA7
Warning:      #FDCB6E
Error:        #FF6B6B
```

## Chart Style

- Line Chart: 연습 트렌드 (시간 경과별)
- Bar Chart: 악기별 연습량 비교
- Donut Chart: 연습 유형 분포
- Radar Chart: 악기별 역량 지표
- Heatmap: 연습 일수 캘린더 (GitHub contribution 스타일)
- Progress Bar: 목표 달성률, 성취도

## Global Components

### 사이드바 (Sidebar)

모든 인증된 페이지에서 접근 가능한 사이드바 메뉴:
- 대시보드
- 연습 시작
- 연습 기록
- **메트로놈** (모달로 표시)
- **튜너** (모달로 표시)
- AI 분석
- 악기 관리
- 설정

### 메트로놈 모달

- 사이드바에서 "메트로놈" 클릭 시 모달 오버레이로 표시
- 전체 페이지를 덮지 않고 중앙에 위치
- 배경 클릭 또는 ESC 키로 닫기
- 모달 내 기능:
  - BPM 조절 (30~300)
  - 재생/정지 버튼
  - 박자 설정 (4/4, 3/4, 6/8 등)
  - 시각적 비트 인디케이터
  - 연습 세션과 독립적으로 작동

### 튜너 모달

- 사이드바에서 "튜너" 클릭 시 모달 오버레이로 표시
- 전체 페이지를 덮지 않고 중앙에 위치
- 배경 클릭 또는 ESC 키로 닫기
- 모달 내 기능:
  - 악기 선택 (기타, 베이스, 우쿨렐레, 바이올린)
  - 현 선택 (선택한 악기의 표준 튜닝)
  - 실시간 음표 감지 및 표시
  - 주파수(Hz) 표시
  - 바늘 게이지로 정확도 시각화 (-50 ~ +50 cents)
  - 색상 코딩: 정확함(녹색), 약간 벗어남(노랑), 많이 벗어남(빨강)
  - 마이크 접근 권한 요청 및 오류 처리
