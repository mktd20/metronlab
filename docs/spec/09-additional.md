# 09. Additional Features (Proposed)

## Social & Community

### Practice Room (실시간 합주 연습방)
- WebRTC 기반 실시간 오디오 스트리밍
- 동일 메트로놈 동기화
- 2~4명 동시 연습
- 방 생성/참여 (초대 링크)

### Practice Feed
- 연습 기록 공유 (선택적 공개)
- 다른 사용자 연습 현황 타임라인
- 응원/리액션 기능
- 팔로우/팔로잉

## Content Library

### Built-in Sheet Music
- 악기별 기본 제공 악보 라이브러리
- 난이도별 분류 (Beginner / Intermediate / Advanced / Expert)
- 장르별 분류
- 검색 기능

### Custom Sheet Upload
- 사용자 악보 업로드 (MusicXML, MIDI, PDF)
- MusicXML/MIDI → 내부 포맷 자동 변환
- 커뮤니티 공유 (선택적)

## Practice Tools

### Tuner (조율기)
- Web Audio API + 마이크 입력
- 크로매틱 튜너
- 악기별 표준 튜닝 프리셋
- 정확도 시각화 (바늘 게이지)

### Chord Dictionary (코드 사전)
- 기타/피아노/우쿨렐레 코드 다이어그램
- 코드 검색
- 코드 진행 제안

### Scale Reference (스케일 레퍼런스)
- 모든 키의 주요 스케일 시각화
- 악기별 포지션 표시
- 스케일 연습 모드 연동

### Drum Pattern Library
- 기본 드럼 패턴 라이브러리 (8-beat, 16-beat, shuffle, etc.)
- 패턴 시각 에디터
- 커스텀 패턴 저장

## Goal & Routine Management

### Practice Goals
- 일간/주간/월간 연습 목표 설정
  - 시간 목표: "하루 30분"
  - 세션 목표: "주 5회"
  - BPM 목표: "이번 달 Flight of the Bumblebee 120 BPM"
- 목표 달성률 대시보드 표시
- 알림 (이메일 / 브라우저 알림)

### Practice Routines
- 루틴 생성: 연습 항목 순서 정의
  - 예: 워밍업(5분) → 스케일(10분) → 곡 연습(20분) → 쿨다운(5분)
- 타이머 기반 자동 전환
- 루틴 템플릿 저장/공유

## Audio Recording & Playback

### Practice Recording
- 마이크 입력으로 연습 녹음
- 세션별 녹음 파일 저장
- 연습 전/후 비교 재생

### Backing Track
- 연습용 배킹 트랙 재생
- 메트로놈과 동기화
- 볼륨 개별 조절 (배킹 / 메트로놈)

## Mobile PWA Support

- PWA (Progressive Web App) 지원
- 오프라인 메트로놈 사용
- 반응형 레이아웃 (모바일 최적화)
- 푸시 알림

## Export & Integration

### Data Export
- 연습 기록 CSV/JSON 내보내기
- 분석 리포트 PDF 생성

### Calendar Integration
- Google Calendar 연동
- 연습 일정 자동 등록
- 리마인더
